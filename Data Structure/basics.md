# Array

## Operations

- lookup O(1)
- push O(1)
- insert O(n)
- delete O(n)

# When to use array?
- Fast lookups
- Fast push/pop
- ordered data

# Why not to use array?

- slow inserts
- slow deletes
- fixed size -- static
